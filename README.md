# Лабораторная работа №2  
_Обобщённая работа с последовательностями на шаблонах_

---

## Постановка задачи  

Реализовать иерархию классов для работы с последовательностями с использованием шаблонов.  
Поддержать основные операции (вставка, удаление, доступ по индексу, фильтрация, отображение, свёртка).  
Обеспечить интерфейс пользователя для работы с различными типами данных (например, `int`, `float`).

---

## Описание решения  
### Используемые типы данных  
- `Sequence<Type>` — абстрактный класс для последовательностей
- `ArraySequence<Type>` — реализация последовательности на динамическом массиве
- `ListSequence<Type>` — реализация последовательности на двусвязном списке
- `Mapper<Type>` — функция, применяемая к каждому элементу (аналог map)
- `Condition<Type>` — функция-фильтр (аналог filter)
- `Reducer<Type>` — функция для свёртки значений (аналог reduce)

## Постановка задачи

Поддержать основные операции (вставка, удаление, доступ по индексу, фильтрация, отображение, свёртка).  
Обеспечить интерфейс пользователя для работы с различными типами данных (например, `int`, `float`).

---

## Описание решения  
### Используемые типы данных  
- `Sequence<Type>` — абстрактный класс для последовательностей  
- `ArraySequence<Type>` — реализация последовательности на динамическом массиве  
- `ListSequence<Type>` — реализация последовательности на двусвязном списке  
- `Mapper<Type>` — функция, применяемая к каждому элементу (аналог map)  
- `Condition<Type>` — функция-фильтр (аналог filter)  
- `Reducer<Type>` — функция для свёртки значений (аналог reduce)  

---

## Функции

### Sequence.hpp

- `virtual Sequence<Type>* Create() const = 0`  
    > Создаёт новую пустую последовательность того же типа  

- `virtual Type& Get(int index) const = 0`  
    > Возвращает элемент по индексу  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **index** — индекс элемента  
    >
    > _ВОЗВРАЩАЕТ:_  
    >   * Ссылка на элемент  

- `virtual int Size() const noexcept = 0`  
    > Возвращает количество элементов  

- `virtual void Append(const Type& item) noexcept = 0`  
    > Добавляет элемент в конец  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **item** — элемент, который нужно добавить  

- `virtual void Prepend(const Type& item) noexcept = 0`  
    > Добавляет элемент в начало  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **item** — элемент, который нужно добавить  

- `virtual void Insert(const Type& item, int index) = 0`  
    > Вставляет элемент по индексу  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **item** — элемент для вставки  
    >   * **index** — позиция, куда вставить  

- `virtual void Remove(int index) = 0`  
    > Удаляет элемент по индексу  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **index** — индекс удаляемого элемента  

- `bool IsEmpty() const`  
    > Проверяет, пуста ли последовательность  
    >
    > _ВОЗВРАЩАЕТ:_  
    >   * true — если пуста, иначе false  

- `Type& GetFirst() const`  
    > Возвращает первый элемент последовательности  

- `Type& GetLast() const`  
    > Возвращает последний элемент последовательности  

- `Sequence<Type>* Map(Mapper<Type> func) const`  
    > Применяет функцию ко всем элементам и возвращает новую последовательность  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **func** — функция для применения к каждому элементу  

- `Sequence<Type>* Where(Condition<Type> filter) const`  
    > Фильтрует элементы по условию  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **filter** — функция-условие для фильтрации  

- `Type Reduce(Reducer<Type> func, const Type& start) const`  
    > Выполняет свёртку (reduce) с начальным значением  
    >
    > _ПАРАМЕТРЫ:_  
    >   * **func** — функция свёртки  
    >   * **start** — начальное значение  
    >
    > _ВОЗВРАЩАЕТ:_  
    >   * Результат свёртки
